# Product

## Register

product

## Users

This is a personal, local-first companion workspace for one user who maintains ongoing AI relationships with multiple characters. The user returns to chat, preserve memories, review echoes, collect keepsakes, and manage API settings without a backend or account system.

## Product Purpose

Vocilege helps the user keep intimate, long-running AI character interactions in a browser-only app. Success means the interface feels trustworthy, emotionally warm, readable, and durable while preserving local data, import/export flows, and real provider API paths.

## Brand Personality

Quiet, intimate, premium. The product should feel like a private sound garden: calm enough for repeated daily use, expressive enough for emotional attachment, and precise enough that data and API controls remain understandable.

## Anti-references

Avoid generic SaaS dashboard theatrics, decorative metrics, unreadable glass effects, gradient text, beige/cream AI-default warmth, dense equal-card walls, and icon-only controls without labels or accessible names. Do not make provider output or thinking content look simulated when a real API path is configured.

## Design Principles

- Emotional clarity before decoration: visual atmosphere must support the user's bond with characters.
- Local-first trust: data, backup, API key, and provider behaviors should remain explicit and predictable.
- Readability is non-negotiable: atmospheric glass and dark themes must still meet contrast requirements.
- Familiar controls over novelty: product UI should use standard buttons, dialogs, settings fields, and focus states.
- Preserve the ritual: "相識 N 天", memories, echoes, keepsakes, and replay features are emotional anchors, not analytics widgets.

## Accessibility & Inclusion

Target WCAG AA contrast for body and muted text. Icon-only controls need accessible labels, titles, keyboard focus, and at least 44px touch targets. Motion should respect reduced-motion preferences, and no critical content should depend on animation or color alone.
